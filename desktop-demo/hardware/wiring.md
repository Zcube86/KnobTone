## 引脚速查

| ESP32 GPIO | 连接 | 说明 |
|-----------|------|------|
| 3V3 | AS5600 VCC | 3.3V 供电 |
| GND | AS5600 GND, TMC6300 GND, 按钮 GND | 共地 |
| GPIO4 | AS5600 SDA | I2C 数据 |
| GPIO5 | AS5600 SCL | I2C 时钟 |
| GPIO6 | TMC6300 IN1 | PWM U 相 |
| GPIO7 | TMC6300 IN2 | PWM V 相 |
| GPIO8 | TMC6300 IN3 | PWM W 相 |
| GPIO9 | TMC6300 EN | 使能 |
| GPIO10 | 按压开关 → GND | 内置 INPUT_PULLUP |
| 5V | TMC6300 VM | 电机电源（建议外接 5V 2A） |

## 电机电源注意事项

- **TMC6300 VM 不要接 ESP32 的 5V 引脚！** ESP32 的 5V 输出电流不足。
- 建议外接 5V 2A 电源（手机充电器 + USB 线即可）。
- BLDC 电机启动瞬间电流大，电源必须够力。

## AS5600 安装

- AS5600 模块需要对准 BLDC 电机的磁铁。
- 磁铁粘贴在电机轴末端，AS5600 固定在电机尾部正上方（间距 1-2mm）。
- 先通电测试：`pio device monitor` 看是否有 `[OK] AS5600 编码器就绪`。

## 按压开关

- 使用微动开关或轻触开关，安装在旋钮帽下方。
- 按压旋钮帽时触发开关。
- 接线：一头接 GPIO10，一头接 GND。
- 代码已启用内部上拉 (`INPUT_PULLUP`)，无需外接电阻。

## 测试步骤

1. **通电前检查**：所有接线无短路
2. **连接 USB**：ESP32 上电
3. **串口监视**：`pio device monitor`，观察初始化输出
4. **旋转测试**：手动旋转旋钮，串口应输出 JSON 旋转事件
5. **按压测试**：按压旋钮，串口应输出按压事件
6. **连接 PC Host**：`python host.py`，旋转旋钮控制系统音量
