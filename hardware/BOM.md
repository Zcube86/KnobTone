## 开发路线

```
V1）BLDC 3pin → TMC6300 → ESP32 sensorless FOC
    目标：旋钮能转、有阻尼感、能区分模式
    
V2）+ AS5600 磁编码器 + 径向磁铁
    目标：4096 步/圈 → 100 个音量 Detent、边界阻力、切歌刻度
    
V3）+ 外壳 + PCB
    目标：从面包板变成可携带的设备
```
